# To do App

## Project setup
```
npm install
```

## Create Env file in root folder
```
Set up env variables for DB example

DB_HOST='127.0.0.1'
DB_PORT='3306'
DB_NAME='tasks'
DB_USER=root
DB_PASS=

```

### Starts server on localhost:3000
```
npm run start
```

### Compiles front end work and minifies for production
```
cd front
npm install
npm run build
```
